# Dockhand Config

Docker-контейнер [Dockhand](https://github.com/typehero/dockhand) с управлением через Make и скриптами резервного копирования.

## Требования

- Docker
- Make

## Настройка

```bash
cp .env.example .env
```

Заполните `.env`:

| Переменная         | Описание                          |
|--------------------|-----------------------------------|
| `HOME_DIR`         | Домашняя директория для монтирования |
| `DOCKHAND_DATA`    | Путь к данным контейнера             |
| `DOCKHAND_BACKUPS` | Путь для хранения резервных копий    |

## Команды

```bash
make run        # Запустить контейнер
make stop       # Остановить контейнер
make restart    # Перезапустить контейнер
make backup     # Создать бэкап директории data
make restore    # Восстановить из последней резервной копии
```
